INTRODUCTION

AI 에이전트란?

스스로 판단하고, 행동하는 AI

AI 에이전트는 단순히 질문에 답하는 AI를 넘어, 스스로 목표를 설정하고, 계획을 세우고, 도구를 사용해 실행하는 자율적 AI 시스템입니다.

01
환경 인식

파일, 웹, API 등 외부 환경을 스스로 탐색하고 정보를 수집

02
자율 판단

수집한 정보를 바탕으로 다음 행동을 스스로 결정

03
도구 활용

코드 실행, 파일 편집, 외부 서비스 호출 등 실제 작업을 수행

04
반복 학습

결과를 확인하고 부족하면 스스로 수정하며 목표 달성까지 반복

AI 에이전트의 핵심은 "자율적 루프"입니다. 사람이 매번 지시하지 않아도 스스로 판단하고 행동합니다.
